# Style Guide

## NHS Identity Guidelines
Attempt as best as possible to stick to the guidelines outlined [here](https://www.england.nhs.uk/nhsidentity/identity-guidelines/colours/).

## Centralisation
Try to keep things like colours, sylesheets, HTML, all in one place to reduce code repetition or style mismatches. This also makes it much easier to change aspects later on, as they only have to be changed in one place.
